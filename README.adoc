= SpringBatchの練習

== 実行

.00-build
----
$ docker-compose build
----

.01-springboot側
----
$ docker-compose run --rm app bash
コンテナ内
$ gradle bootRun
----

.02-db側で確認(別シェル)
----
$ docker-compose exec db bash
コンテナ内
$ psql -U postgres -d app-local
DB内
app-local=# select * from people;

         3 | JUSTIN     | DOE
